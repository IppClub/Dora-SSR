## 三位旅伴，三种性格，三段想象力的延伸

&emsp;&emsp;在《绿野仙踪》中，女孩多萝的旅途并不孤单，而在 Dora SSR 社区的角色世界中，我们也为多萝设计了三位同样个性鲜明的旅伴：**塞拉（Sera）**、**提妮（Tinie）** 和 **若澜（Rola）**。

&emsp;&emsp;他们的形象不仅取材于经典角色，更融入了游戏与开源文化的审美与精神。每一位都是由社区成员共同参与塑造的原创角色，充满活力与创意，背后藏着不止是一张立绘，而是一种性格的描绘、一种象征的延伸、一种想象力的展现。

---

## 吉祥物角色设计（续）

:::info 角色介绍
**塞拉（Sera）** 是一位拥有金色眼睛的稻草少女，左眼被头发遮住，右臂和左腿有缝线。她常穿着打补丁的米色麻布衣，头戴彩色麦穗发饰，动作端庄却有一种别样的坚韧。她的存在象征着“修补的智慧”——不是天赋异禀，却靠日积月累学会思考问题。她用稻草做的四指灵巧地打着代码的结，沉稳、细致、安静如田野的风。
:::

:::info 角色介绍
**提妮（Tinie）** 是一位仿生构造体少女，身穿淡紫与灰银拼色的装甲服，胸口嵌着一颗像素风黑白心形标志。每一次睁大眼睛发呆时，眼角那两条金属状刻痕好像在说：“我还不太懂人类的情绪……但我正在努力计算中！”。她总是在后台悄悄优化引擎性能，是代码世界的守护者，也是创作者最可靠的 bug 消除员。她看起来有点呆，其实却藏着一颗“不让任何游戏帧掉下来”的坚定内核！
:::

:::info 角色介绍
**若澜（Rola）** 是一位性格张扬、劲头十足的幼年狮子王。他总是第一个跳进战场的玩家，也永远是社区活动里最带劲的氛围组！头戴狮耳发箍，眼神中总带着跃跃欲试的光芒。他敢于尝试各种新玩法，是开源游戏创意的点子王，也是诸多玩法测试的“第一实验家”。
:::
