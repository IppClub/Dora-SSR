## 探索开源游戏的艺术想象力！

&emsp;&emsp;游戏开发不仅仅是代码与算法的堆砌，更是一种艺术创作的表达。在 Dora SSR 开源社区，我们不仅关注技术工具的研发，更珍视每一份创造力的迸发。

&emsp;&emsp;在这里，我们也汇聚热爱创作的开发者、设计师、艺术家，共同探索游戏与艺术的交汇点。从开源社区的吉祥物、插画、音乐，到充满个性的周边设计，每一份作品都承载着我们的热情与想象力。

&emsp;&emsp;**SSR 文创**作为社区的文创展示区，将这些创作整理呈现，为每一位热爱游戏艺术的伙伴提供灵感与分享的平台。无论是手绘的角色设定、游戏世界观的视觉演绎，还是轻快悦耳的原创音乐，我们期待与你一起，拓展开源游戏的更多可能。

:::tip 加入我们
&emsp;&emsp;欢迎加入我们，用创意点亮开源，一起来创作属于 Dora SSR 社区的艺术篇章！
:::

## 吉祥物角色设计

:::info 角色介绍
&emsp;&emsp;**多萝（Dora）** 是 Dora SSR 开源游戏引擎的吉祥物，她的形象灵感来自《绿野仙踪》的小主角，但以更具活力和探索精神的设计展现开源精神。她是开发者的伙伴，性格沉稳却又勇敢探索代码世界，陪伴每位创作者开启属于自己的奇幻游戏旅程！
:::
