## 从候选设计到新伙伴

这两个吉祥物最初分别是“狮子（若澜）”与“铁皮女孩（提妮）”的候选设计。随着角色世界逐渐成形，她们鲜明的外形与性格也展现出了属于自己的可能性。

如今，她们已经走出最初的候选设定，正式作为新伙伴加入 Dora SSR 社区。一次角色探索由此延伸出了新的故事，也让每一份创意草稿都有机会成为角色世界中意想不到的新篇章。
