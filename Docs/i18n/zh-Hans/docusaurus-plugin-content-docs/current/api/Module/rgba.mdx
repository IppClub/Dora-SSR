# rgba

**描述：**

&emsp;&emsp;通过RGBA颜色通道值创建一个新的`Color`对象。

**签名：**
```tl
rgba: function(r: number, g: number, b: number, a: number): Color
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| r | number | 红色通道值（0-255）。 |
| g | number | 绿色通道值（0-255）。 |
| b | number | 蓝色通道值（0-255）。 |
| a | number | 透明通道值（0-1）。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Color | 新的`Color`对象。 |
