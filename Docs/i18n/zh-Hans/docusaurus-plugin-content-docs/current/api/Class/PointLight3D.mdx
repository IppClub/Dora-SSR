# PointLight3D

**描述：**

&emsp;&emsp;位置由世界变换决定的点光源。

## color

**类型：** 成员变量。

**描述：**

&emsp;&emsp;sRGB 空间中的灯光颜色。

**签名：**
```tl
color: Color3
```

## intensity

**类型：** 成员变量。

**描述：**

&emsp;&emsp;灯光强度。

**签名：**
```tl
intensity: number
```

## range

**类型：** 成员变量。

**描述：**

&emsp;&emsp;灯光的最大有效范围。

**签名：**
```tl
range: number
```
