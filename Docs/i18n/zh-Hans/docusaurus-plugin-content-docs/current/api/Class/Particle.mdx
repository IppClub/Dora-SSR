# Particle

**描述：**

&emsp;&emsp;用于发射动画粒子的粒子系统场景节点类。

**类对象：**[Particle Class](/docs/api/Class%20Object/Particle)。

**继承自：**[Node](/docs/api/Class/Node)。

## active

**类型：** 只读成员变量。

**描述：**

&emsp;&emsp;粒子系统是否处于活动状态。

**签名：**
```tl
const active: boolean
```

## start

**类型：** 函数。

**描述：**

&emsp;&emsp;开始发射粒子。

**签名：**
```tl
start: function(self: Particle)
```

## stop

**类型：** 函数。

**描述：**

&emsp;&emsp;停止发射粒子，并等待所有活动粒子结束它们的生命周期。

**签名：**
```tl
stop: function(self: Particle)
```

## onFinished

**类型：** 函数。

**描述：**

&emsp;&emsp;注册粒子系统结束时的回调函数。
当粒子系统节点在启动之后又停止发射粒子，并等待所有已发射的粒子结束它们的生命周期时触发。

**签名：**
```tl
onFinished: function(self: Particle, callback: function())
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| callback | function | 粒子系统结束时的回调函数。 |
