# Model3D

**描述：**

&emsp;&emsp;用于渲染 glTF 3D 模型的类。

**继承自：**[Node3D](/docs/api/Class/Node3D)。

## speed

**类型：** 成员变量。

**描述：**

&emsp;&emsp;当前动画的播放速度倍率。

**签名：**
```tl
speed: number
```

## duration

**类型：** 只读成员变量。

**描述：**

&emsp;&emsp;当前动画的时长，单位为秒。

**签名：**
```tl
const duration: number
```

## elapsed

**类型：** 只读成员变量。

**描述：**

&emsp;&emsp;当前动画已经播放的时间，单位为秒。

**签名：**
```tl
const elapsed: number
```

## playing

**类型：** 只读成员变量。

**描述：**

&emsp;&emsp;当前是否正在播放动画。

**签名：**

```tl
const playing: boolean
```

## paused

**类型：** 只读成员变量。

## play

**类型：** 函数。

**描述：**

&emsp;&emsp;按名称播放动画。

**签名：**
```tl
play: function(self: Model3D, name?: string, loop?: boolean --[[false]]): number
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| name | string | [可选] 动画名称。省略时使用默认动画。 |
| loop | boolean | [可选] 动画是否循环播放。默认为 false。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| number | 动画时长，单位为秒。未找到时返回 0。 |
