# JointDef

**描述：**

&emsp;&emsp;定义要创建关节的属性的类。

**类对象：**[JointDef Class](/docs/api/Class%20Object/JointDef)。

**继承自：**[Object](/docs/api/Class/Object)。

## center

**类型：** 成员变量。

**描述：**

&emsp;&emsp;关节的中心点，以本地坐标表示。

**签名：**
```tl
center: Vec2
```

## position

**类型：** 成员变量。

**描述：**

&emsp;&emsp;关节的位置，以世界坐标表示。

**签名：**
```tl
position: Vec2
```

## angle

**类型：** 成员变量。

**描述：**

&emsp;&emsp;关节的角度，以度数表示。

**签名：**
```tl
angle: number
```
