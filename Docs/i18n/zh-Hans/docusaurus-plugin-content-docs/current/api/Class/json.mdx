# json

**描述：**

&emsp;&emsp;Dora的JSON库。

## decode

**类型：** 函数。

**描述：**

&emsp;&emsp;解析指定的JSON文本，并返回相应的表。

**签名：**
```tl
decode: function(json: string, maxDepth?: integer): table | nil, nil | string
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| json | string | 要解析的JSON文本。 |
| maxDepth | number | [可选] 解析的最大深度（默认为128）。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| table \| nil, nil \| string | 表示JSON数据的表，如果文本不是有效的JSON，则返回nil和错误消息。 |

## encode

**类型：** 函数。

**描述：**

&emsp;&emsp;将指定的表转换为JSON文本。

**签名：**
```tl
encode: function(tab: table): string | nil, nil | string
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| tab | table | 要转换的表。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| string \| nil, nil \| string | 表示表的JSON文本，如果无法转储表，则返回nil和错误消息。 |

## null

**类型：** 成员变量。

**描述：**

&emsp;&emsp;表示JSON null值。

**签名：**
```tl
null: JsonNull
```
