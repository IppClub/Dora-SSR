# SearchFilesResult

**描述：**

&emsp;&emsp;`searchFilesAsync` 返回的搜索结果项。

## pos

**类型：** 成员变量。

**描述：**

&emsp;&emsp;匹配位置（字节偏移）。

**签名：**
```tl
pos: integer
```

## line

**类型：** 成员变量。

**描述：**

&emsp;&emsp;匹配所在行号（从 1 开始）。

**签名：**
```tl
line: integer
```

## column

**类型：** 成员变量。

**描述：**

&emsp;&emsp;匹配所在列号（从 1 开始）。

**签名：**
```tl
column: integer
```

## content

**类型：** 成员变量。

**描述：**

&emsp;&emsp;匹配到的内容片段。

**签名：**
```tl
content: string
```
