# Menu

**描述：**

&emsp;&emsp;用于管理一个矩形区域内子节点的触摸事件的场景节点类。
会中断子节点触摸事件的向上传播。
只有第一个触摸事件可以被一个子节点接收到，后来触发的多点触摸事件将被忽略。

**类对象：**[Menu Class](/docs/api/Class%20Object/Menu)。

**继承自：**[Node](/docs/api/Class/Node)。

## enabled

**类型：** 成员变量。

**描述：**

&emsp;&emsp;菜单当前是否启用或禁用。

**签名：**
```tl
enabled: boolean
```
