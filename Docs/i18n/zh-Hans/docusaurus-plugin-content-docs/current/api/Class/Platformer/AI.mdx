# AI

**描述：**

&emsp;&emsp;用于在执行决策树AI时获取信息的单例类。

## getUnitsByRelation

**类型：** 函数。

**描述：**

&emsp;&emsp;获取游戏单位的检测范围内具有指定关系的单位。

**签名：**
```tl
getUnitsByRelation: function(self: AI, relation: Relation): Array
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| relation | Relation | 游戏单位的关系。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Array | 具有指定关系的游戏单位列表。 |
