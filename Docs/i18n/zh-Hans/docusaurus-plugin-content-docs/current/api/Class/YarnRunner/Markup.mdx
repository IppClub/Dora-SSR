# Markup

**描述：**

&emsp;&emsp;定义标记属性的类。

## name

**类型：** 成员变量。

**描述：**

&emsp;&emsp;标记的名称。

**签名：**
```tl
name: string
```

## start

**类型：** 成员变量。

**描述：**

&emsp;&emsp;标记的起始位置。

**签名：**
```tl
start: integer
```

## stop

**类型：** 成员变量。

**描述：**

&emsp;&emsp;标记的结束位置。

**签名：**
```tl
stop: integer
```

## attrs

**类型：** 成员变量。

**描述：**

&emsp;&emsp;与标记相关的属性。

**签名：**
```tl
attrs: {string: boolean | number | string} | nil
```
