# Node3D

**描述：**

&emsp;&emsp;用于构建 3D 对象层级树的类。

**继承自：**[Object](/docs/api/Class/Object)。

**类型：** 成员变量。

**描述：**

&emsp;&emsp;节点在父节点子节点数组中的顺序。

## addChild

**类型：** 函数。

**描述：**

&emsp;&emsp;添加子 3D 节点。

**签名：**
```tl
addChild: function(
		self: Node3D,
		child: Node3D,
		order?: integer --[[0]],
		tag?: string --[[""]]
	)
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| child | Node3D | 要添加的子节点。 |
| order | integer | [可选] 子节点的绘制顺序。默认为 0。 |
| tag | string | [可选] 子节点的标签。默认为空字符串。 |

## removeChild

**类型：** 函数。

**描述：**

&emsp;&emsp;移除子 3D 节点。

**签名：**
```tl

```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| child | Node3D | 要移除的子节点。 |
| cleanup | boolean | [可选] 是否清理子节点。默认为 true。 |
