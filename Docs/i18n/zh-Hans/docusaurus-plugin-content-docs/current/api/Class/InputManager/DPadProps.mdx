# DPadProps

**描述：**

&emsp;&emsp;虚拟方向键（D-pad）的属性。

## width

**类型：** 成员变量。

**描述：**

&emsp;&emsp;D-pad的按钮宽度。[可选]

**签名：**
```tl
width: number
```

## height

**类型：** 成员变量。

**描述：**

&emsp;&emsp;D-pad的按钮高度。[可选]

**签名：**
```tl
height: number
```

## offset

**类型：** 成员变量。

**描述：**

&emsp;&emsp;D-pad的按钮间距。[可选]

**签名：**
```tl
offset: number
```

## inputManager

**类型：** 成员变量。

**描述：**

&emsp;&emsp;D-pad的输入管理器。[必需]

**签名：**
```tl
inputManager: InputManager
```
