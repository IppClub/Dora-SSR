# TriggerPadProps

**描述：**

&emsp;&emsp;虚拟触发器按钮UI的属性。

## buttonSize

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮的大小。[可选]

**签名：**
```tl
buttonSize: number
```

## fontName

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮盘的字体名称。[可选]

**签名：**
```tl
fontName: string
```

## inputManager

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮盘的输入管理器。[必需]

**签名：**
```tl
inputManager: InputManager
```

## color

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮盘的颜色。[可选]

**签名：**
```tl
color: number
```

## primaryOpacity

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮盘的主要透明度。[可选]

**签名：**
```tl
primaryOpacity: number
```

## noShoulder

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏肩部按钮。[可选]

**签名：**
```tl
noShoulder: boolean
```
