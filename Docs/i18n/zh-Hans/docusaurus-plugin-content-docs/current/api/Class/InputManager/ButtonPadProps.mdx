# ButtonPadProps

**描述：**

&emsp;&emsp;虚拟按钮盘UI的属性。

## buttonSize

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮的大小。[可选]

**签名：**
```tl
buttonSize: number
```

## buttonPadding

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮的间距。[可选]

**签名：**
```tl
buttonPadding: number
```

## fontName

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮盘的字体名称。[可选]

**签名：**
```tl
fontName: string
```

## inputManager

**类型：** 成员变量。

**描述：**

&emsp;&emsp;按钮盘的输入管理器。[必需]

**签名：**
```tl
inputManager: InputManager
```
