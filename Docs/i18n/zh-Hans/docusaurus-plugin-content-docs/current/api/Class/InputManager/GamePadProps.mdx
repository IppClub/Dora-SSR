# GamePadProps

**描述：**

&emsp;&emsp;虚拟游戏手柄UI的属性。

## noDPad

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏方向键盘（D-pad）。[可选]

**签名：**
```tl
noDPad: boolean
```

## noLeftStick

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏左摇杆（L、LS）。[可选]

**签名：**
```tl
noLeftStick: boolean
```

## noRightStick

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏右摇杆（R、RS）。[可选]

**签名：**
```tl
noRightStick: boolean
```

## noButtonPad

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏按钮盘（A、B、X、Y）。[可选]

**签名：**
```tl
noButtonPad: boolean
```

## noTriggerPad

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏触发器盘（LB、LT、RB、RT）。[可选]

**签名：**
```tl
noTriggerPad: boolean
```

## noControlPad

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏控制盘（开始和返回按钮）。[可选]

**签名：**
```tl
noControlPad: boolean
```

## noShoulder

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏肩部按钮（LB、RB）。[可选]

**签名：**
```tl
noShoulder: boolean
```

## noStickButton

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏摇杆按钮（LS、RS）。[可选]

**签名：**
```tl
noStickButton: boolean
```

## color

**类型：** 成员变量。

**描述：**

&emsp;&emsp;游戏手柄的颜色。[可选]

**签名：**
```tl
color: number
```

## primaryOpacity

**类型：** 成员变量。

**描述：**

&emsp;&emsp;游戏手柄的主要透明度。[可选]

**签名：**
```tl
primaryOpacity: number
```

## secondaryOpacity

**类型：** 成员变量。

**描述：**

&emsp;&emsp;游戏手柄的次要透明度。[可选]

**签名：**
```tl
secondaryOpacity: number
```

## inputManager

**类型：** 成员变量。

**描述：**

&emsp;&emsp;游戏手柄的输入管理器。[必需]

**签名：**
```tl
inputManager: InputManager
```
