# JoyStickProps

**描述：**

&emsp;&emsp;虚拟操纵杆UI的属性。

## stickType

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的类型。[必需]

**签名：**
```tl
stickType: JoyStickType
```

## moveSize

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的移动大小。[可选]

**签名：**
```tl
moveSize: number
```

## hatSize

**类型：** 成员变量。

**描述：**

&emsp;&emsp;帽子的大小。[可选]

**签名：**
```tl
hatSize: number
```

## fontName

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的字体名称。[可选]

**签名：**
```tl
fontName: string
```

## buttonSize

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆按钮的大小。[可选]

**签名：**
```tl
buttonSize: number
```

## inputManager

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的输入管理器。[必需]

**签名：**
```tl
inputManager: InputManager
```

## color

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的颜色。[可选]

**签名：**
```tl
color: number
```

## primaryOpacity

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的主要透明度。[可选]

**签名：**
```tl
primaryOpacity: number
```

## secondaryOpacity

**类型：** 成员变量。

**描述：**

&emsp;&emsp;操纵杆的次要透明度。[可选]

**签名：**
```tl
secondaryOpacity: number
```

## noStickButton

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否隐藏操纵杆按钮。[可选]

**签名：**
```tl
noStickButton: boolean
```
