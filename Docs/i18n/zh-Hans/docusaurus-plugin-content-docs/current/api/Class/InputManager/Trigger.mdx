# Trigger

**描述：**

&emsp;&emsp;输入触发器的管理类，定义了键盘键、游戏手柄按钮和摇杆的各种输入触发器。

## state

**类型：** 成员变量。

**描述：**

&emsp;&emsp;当前触发器状态。

**签名：**
```tl
state: TriggerState
```

## value

**类型：** 成员变量。

**描述：**

&emsp;&emsp;当前触发器的值。

**签名：**
```tl
value: InputValue
```

## progress

**类型：** 成员变量。

**描述：**

&emsp;&emsp;当前触发器进度，适用时取值范围为0到1。

**签名：**
```tl
progress: number
```

## onChange

**类型：** 函数。

**描述：**

&emsp;&emsp;旧的单个变化回调。需要多个监听器时优先使用 `addListener()`。

**签名：**
```tl
onChange: function()
```

## addListener

**类型：** 函数。

**描述：**

&emsp;&emsp;添加一个监听器，不会替换已有监听器。

**签名：**
```tl
addListener: function(self: Trigger, listener: TriggerListener)
```

## removeListener

**类型：** 函数。

**描述：**

&emsp;&emsp;移除通过 `addListener()` 添加的监听器。

**签名：**
```tl
removeListener: function(self: Trigger, listener: TriggerListener)
```


| input | InputBinding | 输入绑定。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
