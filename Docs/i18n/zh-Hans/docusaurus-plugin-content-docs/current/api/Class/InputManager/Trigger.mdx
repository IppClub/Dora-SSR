# Trigger

**描述：**

&emsp;&emsp;输入触发器的管理类，定义了键盘键、游戏手柄按钮和摇杆的各种输入触发器。

## KeyDown

**类型：** 函数。

**描述：**

&emsp;&emsp;创建一个触发器，当所有指定的键被按下时触发。

**签名：**
```tl
KeyDown: function(combineKeys: KeyName | {KeyName}): Trigger
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| combineKeys | KeyName | \{KeyName} | 要检查的单个键或组合键。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Trigger | 触发器对象。 |

## Block

**类型：** 函数。

**描述：**

**签名：**
```tl

```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| trigger | Trigger | 要被阻塞的触发器。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Trigger | 触发器对象。 |
