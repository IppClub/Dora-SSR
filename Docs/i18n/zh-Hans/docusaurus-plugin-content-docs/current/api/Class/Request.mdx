# Request

**描述：**

&emsp;&emsp;代表一个HTTP请求对象。

## headers

**类型：** 成员变量。

**描述：**

&emsp;&emsp;包含请求头信息的表。

**签名：**
```tl
headers: {string: string}
```

## params

**类型：** 成员变量。

**描述：**

&emsp;&emsp;包含请求参数的表（例如 URL 查询参数）。

**签名：**
```tl
params: {string: string}
```

## body

**类型：** 成员变量。

**描述：**

&emsp;&emsp;请求的内容体。

**签名：**
```tl
body: table | string
```
