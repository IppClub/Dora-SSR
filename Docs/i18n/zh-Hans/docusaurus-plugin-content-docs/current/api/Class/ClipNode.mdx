# ClipNode

**描述：**

&emsp;&emsp;用于给渲染场景设置裁剪蒙版的场景节点类。

**类对象：**[ClipNode Class](/docs/api/Class%20Object/ClipNode)。

**继承自：**[Node](/docs/api/Class/Node)。

## stencil

**类型：** 成员变量。

**描述：**

&emsp;&emsp;定义裁剪形状的蒙版节点。

**签名：**
```tl
stencil: Node
```

## alphaThreshold

**类型：** 成员变量。

**描述：**

&emsp;&emsp;生效的用于裁剪的蒙版像素的alpha阈值。取值范围为0到1。

**签名：**
```tl
alphaThreshold: number
```

## inverted

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否翻转裁剪区域。

**签名：**
```tl
inverted: boolean
```
