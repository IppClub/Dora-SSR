# VideoNode

**描述：**

&emsp;&emsp;用于表示视频节点的类。

**类对象：**[VideoNode Class](/docs/api/Class%20Object/VideoNode)。

**继承自：**[Sprite](/docs/api/Class/Sprite)。

## pause

**类型：** 函数。

**描述：**

&emsp;&emsp;暂停视频播放。

**签名：**
```tl
pause: function(self: VideoNode)
```

## resume

**类型：** 函数。

**描述：**

&emsp;&emsp;继续视频播放。

**签名：**
```tl
resume: function(self: VideoNode)
```

## readonly

**类型：** 成员变量。

**描述：**

&emsp;&emsp;视频当前是否暂停。

**签名：**
```tl
readonly paused: boolean
```
