# Joint

**描述：**

&emsp;&emsp;用于连接物理体的关节类。

**类对象：**[Joint Class](/docs/api/Class%20Object/Joint)。

**继承自：**[Object](/docs/api/Class/Object)。

## world

**类型：** 只读成员变量。

**描述：**

&emsp;&emsp;所属的物理世界。

**签名：**
```tl
const world: PhysicsWorld
```

## destroy

**类型：** 函数。

**描述：**

&emsp;&emsp;将关节从物理世界中移除。

**签名：**
```tl
destroy: function(self: Joint)
```
