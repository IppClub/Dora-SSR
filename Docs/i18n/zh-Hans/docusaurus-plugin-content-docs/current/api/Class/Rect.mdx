## set

**描述：**

**签名：**
```tl

```

**参数：**

| 参数名 | 类型 | 描述 |

| width | number | 矩形的宽度。 |
| height | number | 矩形的高度。 |

## containsPoint

**类型：** 函数。

**描述：**

&emsp;&emsp;检查点是否在矩形内。

**签名：**
```tl

```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| point | Vec2 | 要检查的点，用二维向量对象表示。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| boolean | 点是否在矩形内。 |
