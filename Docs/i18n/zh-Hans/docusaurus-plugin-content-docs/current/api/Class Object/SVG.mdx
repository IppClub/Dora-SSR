# SVG的类对象

**描述：**

&emsp;&emsp;用于创建SVG对象的类。

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;从指定的SVG文件创建一个新的SVG对象。

**签名：**
```tl
metamethod __call: function(self: SVGClass, filename: string): SVG
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| filename | string | SVG格式文件的路径。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| SVG | 创建的SVG对象。 |
