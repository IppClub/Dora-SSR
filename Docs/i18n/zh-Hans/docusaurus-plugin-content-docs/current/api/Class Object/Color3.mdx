# Color3的类对象

**描述：**

&emsp;&emsp;用于创建`Color3`对象的类。

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;从RGB整数值创建新的`Color3`对象。

**签名：**
```tl
metamethod __call: function(self: Color3Class, rgb?: number): Color3
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| rgb | number | [可选] 从中创建颜色的 RGB 整数值。默认值为 0。<br/>例如：0xffffff（白色），0xff0000（红色）。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Color3 | 新的`Color3`对象。 |

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;从RGB颜色通道值创建新的`Color3`对象。

**签名：**
```tl
metamethod __call: function(self: Color3Class, r: number, g: number, b: number): Color3
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| r | number | 红色通道值（0-255）。 |
| g | number | 绿色通道值（0-255）。 |
| b | number | 蓝色通道值（0-255）。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Color3 | 新的`Color3`对象。 |
