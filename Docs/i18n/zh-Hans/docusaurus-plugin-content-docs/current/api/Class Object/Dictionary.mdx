# Dictionary的类对象

**描述：**

&emsp;&emsp;用于创建字典对象的类。

**用法示例：**
```tl
local Dictionary = require("Dictionary")
local dict = Dictionary()
```

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;用于创建字典类型实例的元方法。

**签名：**
```tl
metamethod __call: function(self: DictionaryClass): Dictionary
```

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Dictionary | 字典类型的新实例。 |
