# SVG

**Description:**

&emsp;&emsp;A record used for Scalable Vector Graphics rendering.

**Class Object:** [SVG Class](/docs/api/Class%20Object/SVG).

**Inherits from:** [Object](/docs/api/Class/Object).

## width

**Type:** Readonly Field.

**Description:**

&emsp;&emsp;The width of the SVG object.

**Signature:**
```tl
const width: number
```

## height

**Type:** Readonly Field.

**Description:**

&emsp;&emsp;The height of the SVG object.

**Signature:**
```tl
const height: number
```

## render

**Type:** Function.

**Description:**

&emsp;&emsp;Renders the SVG object, should be called every frame for the render result to appear.

**Signature:**
```tl
render: function(self: SVG)
```
