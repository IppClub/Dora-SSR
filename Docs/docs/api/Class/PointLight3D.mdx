# PointLight3D

**Description:**

&emsp;&emsp;A point light whose position follows its world transform.

## color

**Type:** Field.

**Description:**

&emsp;&emsp;The light color in sRGB space.

**Signature:**
```tl
color: Color3
```

## intensity

**Type:** Field.

**Description:**

&emsp;&emsp;The light intensity.

**Signature:**
```tl
intensity: number
```

## range

**Type:** Field.

**Description:**

&emsp;&emsp;The maximum effective range of the light.

**Signature:**
```tl
range: number
```
