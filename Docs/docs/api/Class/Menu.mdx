# Menu

**Description:**

&emsp;&emsp;This record is used for managing touch events for children nodes in a given area.
Menu will swallow touches that is hitting children nodes.
Only one child node can receive the first touch event, multi-touches comes later for other children node will be ignored.

**Class Object:** [Menu Class](/docs/api/Class%20Object/Menu).

**Inherits from:** [Node](/docs/api/Class/Node).

## enabled

**Type:** Field.

**Description:**

&emsp;&emsp;Whether the menu is currently enabled or disabled.

**Signature:**
```tl
enabled: boolean
```
