## addChild

**Type:** Function.

**Description:**

&emsp;&emsp;Adds a child 3D node to the current node.

**Signature:**
```tl
addChild: function(
		self: Node3D,
		child: Node3D,
		order?: integer --[[0]],
		tag?: string --[[""]]
	)
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| child | Node3D | The child node to add. |
| order | integer | [optional] The drawing order of the child node. Default is 0. |
| tag | string | [optional] The tag of the child node. Default is an empty string. |

## removeChild

**Type:** Function.

**Description:**
