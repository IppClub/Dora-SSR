# SearchFilesResult

**Description:**

&emsp;&emsp;A search result item returned by `searchFilesAsync`.

## pos

**Type:** Field.

**Description:**

&emsp;&emsp;The match position in bytes.

**Signature:**
```tl
pos: integer
```

## line

**Type:** Field.

**Description:**

&emsp;&emsp;The line number of the match (1-based).

**Signature:**
```tl
line: integer
```

## column

**Type:** Field.

**Description:**

&emsp;&emsp;The column number of the match (1-based).

**Signature:**
```tl
column: integer
```

## content

**Type:** Field.

**Description:**

&emsp;&emsp;The matched content snippet.

**Signature:**
```tl
content: string
```
