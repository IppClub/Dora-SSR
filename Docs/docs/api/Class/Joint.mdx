# Joint

**Description:**

&emsp;&emsp;A class that can be used to connect physics bodies together.

**Class Object:** [Joint Class](/docs/api/Class%20Object/Joint).

**Inherits from:** [Object](/docs/api/Class/Object).

## world

**Type:** Readonly Field.

**Description:**

&emsp;&emsp;The physics world that the joint belongs to.

**Signature:**
```tl
const world: PhysicsWorld
```

## destroy

**Type:** Function.

**Description:**

&emsp;&emsp;Destroys the joint and removes it from the physics simulation.

**Signature:**
```tl
destroy: function(self: Joint)
```
