# Model3D

**Description:**

&emsp;&emsp;Class used for rendering a glTF 3D model.

**Inherits from:** [Node3D](/docs/api/Class/Node3D).

## speed

**Type:** Field.

**Description:**

&emsp;&emsp;The playback speed multiplier of the current animation.

**Signature:**
```tl
speed: number
```

## duration

**Type:** Readonly Field.

**Description:**

&emsp;&emsp;The duration of the current animation in seconds.

**Signature:**
```tl

```

**Type:** Readonly Field.

## play

**Type:** Function.

**Description:**

&emsp;&emsp;Plays an animation by name.

**Signature:**
```tl
play: function(self: Model3D, name?: string, loop?: boolean --[[false]]): number
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| name | string | [optional] The animation name. Uses the default animation when omitted. |
| loop | boolean | [optional] Whether the animation should loop. Default is false. |

**Returns:**

| Return Type | Description |
| --- | --- |
| number | The animation duration in seconds, or 0 when not found. |
