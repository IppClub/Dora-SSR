# JointDef

**Description:**

&emsp;&emsp;A record that defines the properties of a joint to be created.

**Class Object:** [JointDef Class](/docs/api/Class%20Object/JointDef).

**Inherits from:** [Object](/docs/api/Class/Object).

## center

**Type:** Field.

**Description:**

&emsp;&emsp;The center point of the joint, in local coordinates.

**Signature:**
```tl
center: Vec2
```

## position

**Type:** Field.

**Description:**

&emsp;&emsp;The position of the joint, in world coordinates.

**Signature:**
```tl
position: Vec2
```

## angle

**Type:** Field.

**Description:**

&emsp;&emsp;The angle of the joint, in degrees.

**Signature:**
```tl
angle: number
```
