# DPadProps

**Description:**

&emsp;&emsp;The virtual directional pad (D-pad) properties.

## width

**Type:** Field.

**Description:**

&emsp;&emsp;The button width of the D-pad. [optional]

**Signature:**
```tl
width: number
```

## height

**Type:** Field.

**Description:**

&emsp;&emsp;The button height of the D-pad. [optional]

**Signature:**
```tl
height: number
```

## offset

**Type:** Field.

**Description:**

&emsp;&emsp;The button padding of the D-pad. [optional]

**Signature:**
```tl
offset: number
```

## inputManager

**Type:** Field.

**Description:**

&emsp;&emsp;The input manager for the D-pad. [required]

**Signature:**
```tl
inputManager: InputManager
```
