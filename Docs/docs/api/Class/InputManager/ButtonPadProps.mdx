# ButtonPadProps

**Description:**

&emsp;&emsp;The virtual button pad properties.

## buttonSize

**Type:** Field.

**Description:**

&emsp;&emsp;The size of the button. [optional]

**Signature:**
```tl
buttonSize: number
```

## buttonPadding

**Type:** Field.

**Description:**

&emsp;&emsp;The padding of the buttons. [optional]

**Signature:**
```tl
buttonPadding: number
```

## fontName

**Type:** Field.

**Description:**

&emsp;&emsp;The font name of the button pad. [optional]

**Signature:**
```tl
fontName: string
```

## inputManager

**Type:** Field.

**Description:**

&emsp;&emsp;The input manager for the button pad. [required]

**Signature:**
```tl
inputManager: InputManager
```
