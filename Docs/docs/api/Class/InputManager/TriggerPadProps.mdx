# TriggerPadProps

**Description:**

&emsp;&emsp;The virtual trigger pad properties.

## buttonSize

**Type:** Field.

**Description:**

&emsp;&emsp;The size of the button. [optional]

**Signature:**
```tl
buttonSize: number
```

## fontName

**Type:** Field.

**Description:**

&emsp;&emsp;The padding of the buttons. [optional]

**Signature:**
```tl
fontName: string
```

## inputManager

**Type:** Field.

**Description:**

&emsp;&emsp;The font name of the trigger pad. [optional]

**Signature:**
```tl
inputManager: InputManager
```

## color

**Type:** Field.

**Description:**

&emsp;&emsp;The color of the trigger pad. [optional]

**Signature:**
```tl
color: number
```

## primaryOpacity

**Type:** Field.

**Description:**

&emsp;&emsp;The primary opacity of the trigger pad. [optional]

**Signature:**
```tl
primaryOpacity: number
```

## noShoulder

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the shoulder buttons (LB, RB). [optional]

**Signature:**
```tl
noShoulder: boolean
```
