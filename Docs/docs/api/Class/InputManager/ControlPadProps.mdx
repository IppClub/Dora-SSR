# ControlPadProps

**Description:**

&emsp;&emsp;The virtual control pad properties.

## buttonSize

**Type:** Field.

**Description:**

&emsp;&emsp;The size of the button. [optional]

**Signature:**
```tl
buttonSize: number
```

## fontName

**Type:** Field.

**Description:**

&emsp;&emsp;The font name of the control pad. [optional]

**Signature:**
```tl
fontName: string
```

## inputManager

**Type:** Field.

**Description:**

&emsp;&emsp;The input manager for the control pad. [required]

**Signature:**
```tl
inputManager: InputManager
```

## color

**Type:** Field.

**Description:**

&emsp;&emsp;The color of the control pad. [optional]

**Signature:**
```tl
color: number
```

## primaryOpacity

**Type:** Field.

**Description:**

&emsp;&emsp;The primary opacity of the control pad. [optional]

**Signature:**
```tl
primaryOpacity: number
```
