# GamePadProps

**Description:**

&emsp;&emsp;The virtual gamepad properties.

## noDPad

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the directional pad (D-pad). [optional]

**Signature:**
```tl
noDPad: boolean
```

## noLeftStick

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the left stick (L, LS). [optional]

**Signature:**
```tl
noLeftStick: boolean
```

## noRightStick

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the right stick (R, RS). [optional]

**Signature:**
```tl
noRightStick: boolean
```

## noButtonPad

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the button pad (A, B, X, Y). [optional]

**Signature:**
```tl
noButtonPad: boolean
```

## noTriggerPad

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the trigger pad (LB, LT, RB, RT). [optional]

**Signature:**
```tl
noTriggerPad: boolean
```

## noControlPad

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the control pad (Start and Back buttons). [optional]

**Signature:**
```tl
noControlPad: boolean
```

## noShoulder

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the shoulder buttons (LB, RB). [optional]

**Signature:**
```tl
noShoulder: boolean
```

## noStickButton

**Type:** Field.

**Description:**

&emsp;&emsp;Whether to hide the trigger buttons (LS, RS). [optional]

**Signature:**
```tl
noStickButton: boolean
```

## color

**Type:** Field.

**Description:**

&emsp;&emsp;The color of the gamepad. [optional]

**Signature:**
```tl
color: number
```

## primaryOpacity

**Type:** Field.

**Description:**

&emsp;&emsp;The primary opacity of the gamepad. [optional]

**Signature:**
```tl
primaryOpacity: number
```

## secondaryOpacity

**Type:** Field.

**Description:**

&emsp;&emsp;The secondary opacity of the gamepad. [optional]

**Signature:**
```tl
secondaryOpacity: number
```

## inputManager

**Type:** Field.

**Description:**

&emsp;&emsp;The input manager for the gamepad. [required]

**Signature:**
```tl
inputManager: InputManager
```
