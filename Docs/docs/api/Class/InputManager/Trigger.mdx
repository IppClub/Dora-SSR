# Trigger

**Description:**

&emsp;&emsp;`Trigger` is a record that defines various input triggers for keyboard keys, gamepad buttons, and joysticks.

## KeyDown

**Type:** Function.

**Description:**

&emsp;&emsp;Create a trigger that triggers when all of the specified keys are pressed down.

**Signature:**
```tl
KeyDown: function(combineKeys: KeyName | {KeyName}): Trigger
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| combineKeys | KeyName | \{KeyName} | The key or combined keys to be checked. |

**Returns:**

| Return Type | Description |
| --- | --- |
| Trigger | The trigger object. |
