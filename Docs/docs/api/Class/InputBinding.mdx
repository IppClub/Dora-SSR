# InputBinding

**Description:**

&emsp;&emsp;A keyboard or gamepad binding for generic trigger helpers.

## button

**Type:** Field.

**Description:**

&emsp;&emsp;A gamepad button or button combination to bind. [optional]

**Signature:**
```tl
button: ButtonName | {ButtonName}
```

## controllerId

**Type:** Field.

**Description:**

&emsp;&emsp;The ID of the gamepad controller. Default is 0. [optional]

**Signature:**
```tl
controllerId: number
```
