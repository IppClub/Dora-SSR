# rgba

**Description:**

&emsp;&emsp;Creates a new `Color` object using the RGBA color channel values.

**Signature:**
```tl
rgba: function(r: number, g: number, b: number, a: number): Color
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| r | number | The red channel value (0-255). |
| g | number | The green channel value (0-255). |
| b | number | The blue channel value (0-255). |
| a | number | The alpha channel value (0-1). |

**Returns:**

| Return Type | Description |
| --- | --- |
| Color | The new `Color` object. |
