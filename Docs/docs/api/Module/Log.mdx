# Log

**Description:**

&emsp;&emsp;Logs a message to the console.

**Signature:**
```tl
Log: function(level: LogLevel, msg: string)
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| level | "Info" | "Warn" | "Error" | The logging level. |
| msg | string | The message to be logged. |
