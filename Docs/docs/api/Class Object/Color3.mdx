# Color3 Class

**Description:**

&emsp;&emsp;A class for creating Color3 objects.

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a color with all channels set to 0.

**Signature:**
```tl
metamethod __call: function(self: Color3Class): Color3
```

**Returns:**

| Return Type | Description |
| --- | --- |
| Color3 | A new `Color3` object. |

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a new `Color3` object from an RGB integer value.

**Signature:**
```tl
metamethod __call: function(self: Color3Class, rgb: integer): Color3
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| rgb | integer | The RGB integer value to create the color from.<br/>For example 0xffffff (white), 0xff0000 (red) |

**Returns:**

| Return Type | Description |
| --- | --- |
| Color3 | A new `Color3` object. |

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a new `Color3` object from RGB color channel values.

**Signature:**
```tl
metamethod __call: function(self: Color3Class, r: integer, g: integer, b: integer): Color3
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| r | integer | The red channel value (0-255). |
| g | integer | The green channel value (0-255). |
| b | integer | The blue channel value (0-255). |

**Returns:**

| Return Type | Description |
| --- | --- |
| Color | A new `Color3` object. |
