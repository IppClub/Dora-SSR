# Pass Class

**Description:**

&emsp;&emsp;A class for creating Pass objects.

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;A metamethod that allows you to create a new Pass object.

**Signature:**
```tl
metamethod __call: function(self: PassClass, vertShader: string, fragShader: string): Pass
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| vertShader | string | The vertex shader file string. |
| fragShader | string | The fragment shader file string.<br/>A shader file string must be one of the formats:<br/>	builtin: + theBuiltinShaderName<br/>	shader_compiled_file.bin<br/>	Shader/shader_source_file.sc<br/>Details:<br/>	1. `"builtin:" + name` loads an embedded built-in shader.<br/>	2. For `.sc` files, the given path is loaded as shader source and compiled immediately.<br/>	3. For `.bin` files, if the given path exists, it is loaded directly.<br/>	4. Otherwise the engine tries `renderer_dir/filename.bin`, where `renderer_dir`<br/>	   depends on the active backend, such as `dx11`, `metal`, `glsl`, `essl`, or `spirv`. |

**Returns:**

| Return Type | Description |
| --- | --- |
| Pass | A new Pass object. |
