# Color Class

**Description:**

&emsp;&emsp;Provides methods for creating Color objects.

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a color with all channels set to 0.

**Signature:**
```tl
metamethod __call: function(self: ColorClass): Color
```

**Returns:**

| Return Type | Description |
| --- | --- |
| Color | A new Color object. |

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a new Color object with a Color3 object and alpha value.

**Signature:**
```tl
metamethod __call: function(self: ColorClass, color: Color3, a: integer): Color
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| color | Color3 | The color as a Color3 object. |
| a | integer | [optional] The alpha value of the color ranging from 0 to 255. |

**Returns:**

| Return Type | Description |
| --- | --- |
| Color | A new Color object. |
