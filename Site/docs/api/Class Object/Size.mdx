# Size Class

**Description:**

&emsp;&emsp;A class for creating Size objects.

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Create a new Size object with the given width and height.

**Signature:**
```tl
metamethod __call: function(
		self: SizeClass,
		width?: number --[[0]],
		height?: number --[[0]]
	): Size
```

**Usage:**
```tl
local size = Size(10, 20)
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| width | number | [optional] The width of the new size (default 0). |
| height | number | [optional] The height of the new size (default 0). |

**Returns:**

| Return Type | Description |
| --- | --- |
| Size | The new Size object. |

## __call
