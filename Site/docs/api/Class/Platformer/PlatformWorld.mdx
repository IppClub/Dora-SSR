# PlatformWorld

**Description:**

&emsp;&emsp;A class representing a 2D platformer game world with physics simulations.

**Inherits from:** PhysicsWorld.

## camera

**Type:** Field.

**Description:**

&emsp;&emsp;The camera used to control the view of the game world.

**Signature:**
```tl
const camera: PlatformCamera
```

## moveChild

**Type:** Function.

**Description:**

&emsp;&emsp;Moves a child node to a new order for a different layer.

**Signature:**
```tl

```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| child | Node | The child node to be moved. |
| newOrder | integer | The new order of the child node. |
