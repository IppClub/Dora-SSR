# Behavior

**Description:**

&emsp;&emsp;A behavior tree framework for creating game AI structures.

## Seq

**Type:** Function.

**Description:**

&emsp;&emsp;Creates a new sequence node that executes an array of child nodes in order.

**Signature:**
```tl
Seq: function(nodes: {Leaf}): Leaf
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| nodes | {Leaf} | An array of child nodes. |

**Returns:**

| Return Type | Description |
| --- | --- |
| Leaf | A new sequence node. |

**Type:** Function.

**Description:**

&emsp;&emsp;Creates a new condition node that executes a check handler function when executed.

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| name | string | The name of the condition. |

| Return Type | Description |
| --- | --- |
| Leaf | A new condition node. |
