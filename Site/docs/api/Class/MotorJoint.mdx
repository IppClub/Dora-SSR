# MotorJoint

**Description:**

&emsp;&emsp;A joint that applies a rotational or linear force to a physics body.

**Inherits from:** [Joint](/docs/api/Class/Joint).

## enabled

**Type:** Field.

**Description:**

&emsp;&emsp;Whether or not the motor joint is enabled.

**Signature:**
```tl
enabled: boolean
```

## force

**Type:** Field.

**Description:**

&emsp;&emsp;The force applied to the motor joint.

**Signature:**
```tl
force: number
```

## speed

**Type:** Field.

**Description:**

&emsp;&emsp;The speed of the motor joint.

**Signature:**
```tl
speed: number
```
