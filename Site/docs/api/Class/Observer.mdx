# Observer

**Description:**

&emsp;&emsp;A record representing an observer of entity changes in the game systems.

## watch

**Type:** Function.

**Description:**

&emsp;&emsp;Watches the components changes to entities that match the observer's component filter.

**Signature:**
```tl
watch: function(self: Observer, func: function(Entity)): Observer
```

**Parameters:**

| Parameter | Type | Description |
| --- | --- | --- |
| func | function | The function to call when a change occurs. |

**Returns:**

| Return Type | Description |
| --- | --- |
| Observer | The same observer, for method chaining. |
