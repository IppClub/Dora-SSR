# Delay

**描述：**

&emsp;&emsp;创建一个动作定义对象，用于延迟动作执行的时间。

**签名：**
```tl
Delay: function(duration: number): ActionDef
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| duration | number | 延迟的持续时间（以秒为单位）。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| ActionDef | 新的动作定义对象。 |
