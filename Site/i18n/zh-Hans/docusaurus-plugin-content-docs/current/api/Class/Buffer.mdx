# Buffer

**描述：**

&emsp;&emsp;用于ImGui输入组件的字符串缓冲区类。

**继承自：** Object。

## size

**类型：** 成员变量。

**描述：**

&emsp;&emsp;缓冲区的大小。

**签名：**
```tl
const size: integer
```

## resize

**类型：** 函数。

**描述：**

&emsp;&emsp;更改缓冲区的大小。

**签名：**
```tl
resize: function(self: Buffer, size: integer)
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| size | integer | 缓冲区的新大小。 |

## zeroMemory

**类型：** 函数。

**描述：**

&emsp;&emsp;将缓冲区中的所有字节设置为零。

**签名：**
```tl
zeroMemory: function(self: Buffer)
```

## toString

**类型：** 函数。

**描述：**

&emsp;&emsp;将缓冲区转换为字符串。

**签名：**
```tl
toString: function(self: Buffer): string
```

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| string | 缓冲区内容的字符串表示形式。 |

## setString

**类型：** 函数。

**描述：**

&emsp;&emsp;使用字符串设置缓冲区的内容。

**签名：**
```tl
setString: function(str: string)
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| str | string | 缓冲区的新内容。 |
