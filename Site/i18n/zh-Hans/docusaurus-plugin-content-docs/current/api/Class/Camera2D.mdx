# Camera2D

**描述：**

&emsp;&emsp;游戏引擎中的2D相机对象类。

**类对象：**[Camera2D Class](/docs/api/Class%20Object/Camera2D)。

**继承自：**[Camera](/docs/api/Class/Camera)。

## rotation

**类型：** 成员变量。

**描述：**

&emsp;&emsp;相机旋转角度（以度为单位）。

**签名：**
```tl
rotation: number
```

## zoom

**类型：** 成员变量。

**描述：**

&emsp;&emsp;缩放相机的缩放参数。如果设置为1.0，则视野内的物件为正常大小。如果设置为2.0，则物件显示为两倍大小。

**签名：**
```tl
zoom: number
```

## position

**类型：** 成员变量。

**描述：**

&emsp;&emsp;相机在游戏世界中的位置。

**签名：**
```tl
position: Vec2
```
