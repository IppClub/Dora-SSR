# View

**描述：**

&emsp;&emsp;用于访问当前应用渲染视图设置的类。

## size

**类型：** 成员变量。

**描述：**

&emsp;&emsp;视图的像素尺寸。

**签名：**
```tl
const size: Size
```

## standardDistance

**类型：** 成员变量。

**描述：**

&emsp;&emsp;视图平面距原点的标准距离。

**签名：**
```tl
const standardDistance: number
```

## aspectRatio

**类型：** 成员变量。

**描述：**

&emsp;&emsp;视图的宽高比。

**签名：**
```tl
const aspectRatio: number
```

## nearPlaneDistance

**类型：** 成员变量。

**描述：**

&emsp;&emsp;近裁剪平面的距离。

**签名：**
```tl
nearPlaneDistance: number
```

## farPlaneDistance

**类型：** 成员变量。

**描述：**

&emsp;&emsp;远裁剪平面的距离。

**签名：**
```tl
farPlaneDistance: number
```

## fieldOfView

**类型：** 成员变量。

**描述：**

&emsp;&emsp;视图的视野（以度数表示）。

**签名：**
```tl
fieldOfView: number
```

## scale

**类型：** 成员变量。

**描述：**

&emsp;&emsp;视图的缩放比例系数。

**签名：**
```tl
scale: number
```

## postEffect

**类型：** 成员变量。

**描述：**

&emsp;&emsp;要应用到视图的后处理着色器特效。

**签名：**
```tl
postEffect: SpriteEffect
```

## vsync

**类型：** 成员变量。

**描述：**

&emsp;&emsp;是否启用垂直同步。

**签名：**
```tl
vsync: boolean
```
