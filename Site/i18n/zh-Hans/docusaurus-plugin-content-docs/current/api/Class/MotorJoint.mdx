# MotorJoint

**描述：**

&emsp;&emsp;用于对物理体施加旋转或线性力的连接关节。

**继承自：**[Joint](/docs/api/Class/Joint)。

## enabled

**类型：** 成员变量。

**描述：**

&emsp;&emsp;关节发动机是否启用。

**签名：**
```tl
enabled: boolean
```

## force

**类型：** 成员变量。

**描述：**

&emsp;&emsp;关节发动机要施加的力。

**签名：**
```tl
force: number
```

## speed

**类型：** 成员变量。

**描述：**

&emsp;&emsp;关节发动机的速度。

**签名：**
```tl
speed: number
```
