# GSlot

**描述：**

&emsp;&emsp;全局信号槽对象，可用于处理全局事件。

**继承自：** Object。

## name

**类型：** 成员变量。

**描述：**

&emsp;&emsp;全局信号槽的名称。

**签名：**
```tl
const name: string
```

## enabled

**类型：** 成员变量。

**描述：**

&emsp;&emsp;该信号槽当前是否启用或禁用。

**签名：**
```tl
enabled: boolean
```
