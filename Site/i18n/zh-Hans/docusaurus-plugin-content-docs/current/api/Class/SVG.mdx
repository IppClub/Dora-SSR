# SVG

**描述：**

&emsp;&emsp;用于渲染可缩放矢量图形的类。

**继承自：** Object。

## width

**类型：** 成员变量。

**描述：**

&emsp;&emsp;SVG对象的宽度。

**签名：**
```tl
const width: number
```

## height

**类型：** 成员变量。

**描述：**

&emsp;&emsp;SVG对象的高度。

**签名：**
```tl
const height: number
```

## render

**类型：** 函数。

**描述：**

&emsp;&emsp;渲染SVG对象，需要在每一帧调用以持续显示渲染结果。

**签名：**
```tl
render: function(self: SVG)
```
