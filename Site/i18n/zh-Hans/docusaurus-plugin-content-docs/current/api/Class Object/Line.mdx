# Line的类对象

**描述：**

&emsp;&emsp;用于创建线条节点对象的类。

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;创建并返回一个新的线条节点对象。

**签名：**
```tl
metamethod __call: function(
		self: LineClass,
		verts: {Vec2},
		color?: Color --[[0xffffffff]]
	): Line
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| verts | {Vec2} | 要添加到线条的顶点表。 |
| color | Color | [可选] 线条的颜色（默认为不透明白色）。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Line | 线条节点对象。 |

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;创建并返回一个新的空线条节点对象。

**签名：**
```tl
metamethod __call: function(self: LineClass): Line
```

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Line | 线条节点对象。 |
