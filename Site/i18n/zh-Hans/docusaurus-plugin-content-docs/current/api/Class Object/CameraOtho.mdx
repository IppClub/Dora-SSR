# CameraOtho的类对象

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;使用给定名称创建一个新的正交相机对象。

**签名：**
```tl
metamethod __call: function(
		self: CameraOthoClass,
		name?: string --[[""]]
	): CameraOtho
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| name | string | 正交相机对象的名称。默认为空字符串。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| CameraOtho | 正交相机对象的新实例。 |
