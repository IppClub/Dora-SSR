# Visual的类对象

**描述：**

&emsp;&emsp;用于创建可视效果对象的类。

## __call

**类型：** 元方法。

**描述：**

&emsp;&emsp;使用指定的名称创建新的可视效果对象。

**签名：**
```tl
metamethod __call: function(self: VisualClass, name: string): Visual
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| name | string | 新可视效果对象的名称。可以是粒子文件、帧动画文件或图像文件。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Visual | 新的可视效果对象。 |
