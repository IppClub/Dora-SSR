# Menu Class

**Description:**

&emsp;&emsp;A class for creating Menu objects.

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a new instance of `Menu` with the specified width and height.
-@param width (number) The width of Menu node.
-@param height (number) The height of Menu node.
-@return (Menu) A new Menu node object.

**Signature:**
```tl
metamethod __call: function(self: MenuClass, width: number, height: number): Menu
```

## __call

**Type:** Metamethod.

**Description:**

&emsp;&emsp;Creates a new instance of `Menu` with 0 width and 0 height.
A menu with zero size will handle full screen touches for children nodes.

**Signature:**
```tl
metamethod __call: function(self: MenuClass): Menu
```

**Returns:**

| Return Type | Description |
| --- | --- |
| Menu | A new Menu node object. |
