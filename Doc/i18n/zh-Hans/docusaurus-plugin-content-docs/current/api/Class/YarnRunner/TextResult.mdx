# TextResult

**描述：**

&emsp;&emsp;定义带有可选标记的文本结果的类。

## text

**类型：** 成员变量。

**描述：**

&emsp;&emsp;文本内容。

**签名：**
```tl
text: string
```

## marks

**类型：** 成员变量。

**描述：**

&emsp;&emsp;文本的可选标记。

**签名：**
```tl
marks: nil | {Markup}
```

## optionsFollowed

**类型：** 成员变量。

**描述：**

&emsp;&emsp;表示下一个叙事推进得到的结果将为选项的标志。

**签名：**
```tl
optionsFollowed: nil | boolean
```
