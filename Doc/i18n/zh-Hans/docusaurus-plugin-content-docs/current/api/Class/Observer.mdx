# Observer

**描述：**

&emsp;&emsp;游戏ECS系统实体更改观察器的类。

**类对象：**[Observer Class](/docs/api/Class%20Object/Observer)。

## watch

**类型：** 函数。

**描述：**

&emsp;&emsp;监控观察器观察的实体上的组件变化，在发生变更时调用处理函数。

**签名：**
```tl
watch: function(self: Observer, func: function(Entity): boolean): Observer
```

**参数：**

| 参数名 | 类型 | 描述 |
| --- | --- | --- |
| func | function | 当发生组件变更时要调用的处理函数，当该处理函数返回 true 时，停止监听组件变化。 |

**返回值：**

| 返回类型 | 描述 |
| --- | --- |
| Observer | 当前的观察器，用于方法级连调用。 |
